### Блоки кода

Используйте блоки кода для отображения многострочных фрагментов с подсветкой синтаксиса. Блоки кода необходимы для наглядного представления примеров кода.

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

#code
````mdc
```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
````
::

При написании блока кода можно указать имя файла, которое будет отображаться над блоком. Иконка автоматически определяется по расширению или имени файла.
Имена файлов помогают пользователям понять расположение и назначение кода в проекте.

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

#code
````mdc
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
````
::

::tip
Некоторые иконки уже определены по умолчанию, но вы можете добавить дополнительные в `app.config.ts` через ключ `ui.prose.codeIcon`:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    prose: {
      codeIcon: {
        terminal: 'i-ph-terminal-window-duotone'
      }
    }
  }
})
```
::

Каждый блок кода имеет встроенную кнопку копирования, которая копирует код в буфер обмена.

::tip
Иконку можно изменить в `app.config.ts` через ключи `ui.icons.copy` и `ui.icons.copyCheck`:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    icons: {
      copy: 'i-lucide-copy',
      copyCheck: 'i-lucide-copy-check'
    }
  }
})
```
::

#### Выделение строк

Чтобы выделить строки кода, добавьте `{}` с номерами строк, которые нужно выделить.
Выделение строк полезно для акцентирования внимания на важных частях примеров кода.

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```ts [nuxt.config.ts]{4-5}
export default defineAppConfig({
  ui: {
    icons: {
      copy: 'i-lucide-copy',
      copyCheck: 'i-lucide-copy-check'
    }
  }
})
```

::code-preview{class="[&>div]:*:my-0 [&>div]:*:w-full"}

    modules: ['@nuxt/ui'],

    css: ['~/assets/css/main.css']
