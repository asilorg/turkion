## Изображение

### Через Markdown

Отображайте изображения с помощью стандартного синтаксиса Markdown. Простой способ вставки изображений.

::code-preview
![Nuxt Social Image](https://nuxt.com/new-social.jpg)

#code
```mdc
![Nuxt Social Image](https://nuxt.com/new-social.jpg)
```
::

::note
Если установлен [`@nuxt/image`](https://image.nuxt.com/get-started/installation), вместо нативного тега `img` будет использоваться компонент `<NuxtImg>`.
::

### Через вставки

Используйте вставки для большей гибкости при работе с изображениями. Вставки предоставляют больший контроль над атрибутами изображения, такими как размер и стилизация.

::code-preview
![Nuxt Social Image](https://nuxt.com/new-social.jpg){height="150"}



#code
```mdc
:img{src="https://nuxt.com/new-social.jpg" alt="Nuxt Social Image" height="150"}
```
::
